# WIP Vehicle creation tool for [MinecraftTransportSimulator](https://github.com/DonBruce64/MinecraftTransportSimulator)

## Building
```bash
git clone https://github.com/sharpoff/vehicle-tool.git --recursive
cd vehicle-tool/ && ./build.sh
```

Now you can run ./build/main

## Dependencies
* glad
* glm
* imgui
* SDL3
* stb
* tiny_obj_loader
* cgltf
