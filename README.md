# WIP Vehicle creation tool for [MinecraftTransportSimulator](https://github.com/DonBruce64/MinecraftTransportSimulator)

## Building
```bash
git clone https://github.com/sharpoff/vehicle-tool.git --recursive
cd vehicle-tool/ && ./build.sh
```

Now you can run ./build/main

## Key bindings
* WASD -> move camera
* LMB + mouse move -> rotate camera around
* P -> live reload shaders
* F -> switch fullscreen mode
* T -> translation gizmo
* R -> rotation gizmo
* E -> scale gizmo

## Dependencies
* glad
* glm
* imgui
* Imguizmo
* SDL3
* stb
* tiny_obj_loader
* cgltf
